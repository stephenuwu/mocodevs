Information

Montgomery County Student Developers is a Discord server where you can talk about developing, get help on projects, and claim code bounties.

Code bounties are challenges you can complete, varying in difficulty, and earn real-life money. The money given out will be in cryptocurrency but you don't need a wallet.

Join Server

Mission

Our mission is to connect developers in the MoCo area. We want to create a place where everyone can find help and talk with others.

MoCo Devs should be a place where everyone, regardless of skill or age, feels comfortable and finds enjoyment.

Vision

Our vision is to provide a place for anyone to seek help and discuss anything with other people. MoCo devs should be a place where people enjoy being at.

We also want to introduce coding to more people. We are trying to release the stigma around coding that you have to be extremely nerdy or smart to be good at it or to find it fun.
